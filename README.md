# Multivariate-time-series-prediction

This repository contains the iPython notebook on multivariate time forecasting using LSTM in keras. The dataset used is air pollution forecasting. Model predicts the polluion at time t+1 when given time t.

# Requirments
1. Keras
2. Matplotlib
3. Pandas
4. Numpy
5. Sklearn
